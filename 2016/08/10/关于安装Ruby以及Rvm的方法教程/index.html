关于安装Ruby以及Rvm的方法教程

Aug 10, 2016

关于安装CocoaPods大家遇到最多的问题就是需要ruby的版本>=2.2.2,然后大家跑去安装ruby,发现ruby是依赖于其他的系统库文件,想要安装ruby,就要先安装rvm,坑爹啊 可是我们Mac os系统只是自带了ruby却没有自带rvm,下面说说如何进行安装rvm还有ruby

RVM 是一个命令行工具，可以提供一个便捷的多版本 Ruby 环境的管理和切换

拷贝以下命令到终端中进行安装,根据操作进行

  • 安装RVM

curl -sSL https://get.rvm.io | bash -s stable

  • 列出已知的Ruby版本

rvm list known

  • 安装一个指定的Ruby版本(这里指定的版本为2.3)

rvm install 2.3

  • 设置系统默认版本(这样一来以后新打开的控制台默认的 Ruby 就是这个版本)

rvm use 2.3.0 --default

  • 查询已经安装的Ruby版本信息

rvm list

  • 卸载一个指定的安装的版本

rvm remove 2.3

接下来我们要做什么?当然是检查一下是否都安装成功了!

  • 检测安装的RVM版本(也可以验证是否安装成功,如果不成功.则会提示命令不存在)

rvm -v

ruby -v
