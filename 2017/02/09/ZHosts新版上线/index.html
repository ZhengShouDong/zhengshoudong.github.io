ZHosts新版上线

Feb 9, 2017

新年过完了,还有不到一年的时间又要过年了.提前恭祝大家狗年快乐(微笑脸)

本次更新内容:

  1. 软件主体好久没更新,系统后台可是一天都没歇过.修复了N个Bug,提升用户体验.
  2. 修复某些情况下,软件一开启就闪退的问题.
  3. 完善用户通知,更缓和,降低侵入性.
  4. 图标和UI的改版,感谢:@A_7rr.
  5. 后台服务器的完整更新,修复之前的某些地区网络问题导致的软件不能正常使用.现在的服务器解析和下载更快!

这是2017年发的第一个版本,新的一年会更加的多多完善.网络自由是我们大家的,新的一年我们’一七’努力
